# heroicons-lwc

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

[LWC](https://lwc.dev/) component for [Heroicons](https://heroicons.com/).

## Installation

Install the `heroicons-lwc` package in your project with `yarn` or `npm`.

```bash
// with npm
npm install heroicons-lwc

// with yarn
yarn add heroicons-lwc
```

Or copy the component source (`src/icons/heroicons`) to your local project.

## Usage

component.html

```html
<template>
  <icons-heroicons icon-name="x_circle_outline"></icons-heroicons>
</template>
```

> Note the syntax for icon names uses underscores (instead of dashes) followed by the type (outline or solid) separated by an underscore.

## Building

To build the latest version of Heroicons from source:

```bash
npm run build:icons
```
